== Description ==

記事をいつ書いたかを分析するプラグインです。下記のような分類で記事数などの表を表示します。詳しい説明は http://accountingse.net/2013/02/638/ に書いてます。
* 曜日毎
* 時刻毎
* 平日休日毎
* 曜日・時刻毎
* 平日休日・時刻毎
* 記事作成にかけた時間
* 生データ（table）
* 生データ（csv）

This is plug-in which analyzes when posts were written. Tables, such as the number of posts, are displayed by the following classifications. 
* by day of week
* by hour
* by holiday or not
* by day of week and hour
* by holiday or not and hour
* by spent time
* raw data (table)
* raw data (csv)

== Installation ==

1. Upload `itsukaita` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
